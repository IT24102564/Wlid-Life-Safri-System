Wildlife Safari System
Project Description
A complete Web-based Wildlife Safari Tour Management System developed as a 2nd year university  project. This system to Fix to manual system problem, Provide online booking,Automate jeep and guide assignment,Improve jeep and guide assignemnt,Improve communication, Improve report and decission making.
Features  : Online Safari Booking ,Jeep and Guide Scheduling ,User Login and Roles,Booking Confirmation and Notifications,Daily Reports , Maintenance Requests ,Tourist Profile Management , Feedback and Rating System , Admin Panel for System Management .
Technical Scope:
 Frontend: Responsive web interface with eye-catching interface
(HTML5, CSS, JavaScript).
 Backend: Java with MySQL database.
 Security: Role-based authentication.
